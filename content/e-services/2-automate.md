### Automate

Use automation to simplify life.

We are strong advocates of DevOps Culture. We practice Continuous Integration, Continuous Delivery and Distributed Development.

We believe these practices result in fast, scalable and focused value delivery to stakeholders, while ensuring high quality systems.

Tools we have used for automation: 

- Bash Scripts, Linux Cron Scheduler
- Python Scripts, Ruby Scripts, JavaScript on Node.js
- Git, Jenkins
- Terraform, Ansible, and 
Apache Airflow
